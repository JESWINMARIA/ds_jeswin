# ds_jeswin

This repository contains my submission for the Web3 Trading Team Data Science Assignment.

## 📁 Directory Structure
```
ds_jeswin/
├── notebook_1.ipynb
├── csv_files/
│   ├── historical_data.csv

├── outputs/
│   ├── avg_pnl_chart.png
│   ├── buy_sell_chart.png
│   ├── win_rate_chart.png
├── ds_report.pdf
└── README.md
```

## 🚀 How to Use This Repository

1. Open **notebook_1.ipynb** in Google Colab.
2. Upload the CSVs from the `csv_files/` folder if needed.
3. Run all cells from top to bottom.
4. All charts will automatically be generated and saved into the `output/` directory:
   - `avg_pnl_chart.png`
   - `buy_sell_chart.png`
   - `win_rate_chart.png`
5. A final PDF summary of insights is included as **ds_report.pdf**.
